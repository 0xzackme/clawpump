## Environment Variable Updates

Add this to your `.env`:

```bash
# Whitelist for wallets that can bypass 2M $CLAWDPUMP requirement
# Comma-separated list of Solana wallet addresses
LAUNCH_WHITELIST=YourWallet1Here,YourWallet2Here

# Solana RPC URL (optional, defaults to public mainnet RPC)
SOLANA_RPC_URL=https://api.mainnet-beta.solana.com
```
